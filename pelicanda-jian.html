使用pelican搭建主页

pelican介绍

pelican是一个使用python实现的静态网页生成器。可以搭配GitHubPages作为个人博客使用。

准备
  1. 安装python
  2. 安装pip1
  3. 安装pelican
  4. 安装markdown
过程
  1. 创建一个文件目录，进入并运行

pelican-quickstart 根据提示快速构建一个pelican的文件系统。

  1. 完成后会生成相应的文件（windows系统不支持make命令）。

  2. 可以从Github上获取喜欢的主题，拉取并配置。

git clone https://github.com/getpelican/pelican-themes
